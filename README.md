# Gestion des notes - DEVOIR PROG 2

## Description : 
Ce projet est réalisé dans le cadre de PROG2 et a pour objectif de modéliser un système de gestion des notes dans un établissement scolaire.

## Prérequis :
* Java 17 ou supérieur
* Maven 3.x
* IntelliJ IDEA ou un autre IDE Java
* Git pour cloner le dépôt

## Installation et exécution :
1. Cloner le Dépôt
  ``` bash
    git clone https://github.com/Ery2577/Devoir-PROG2.git
    cd Devoir-PROG2
  ```

2. Construire le projet
``` bash
     mvn clean install
```

3. Lancer l'application 
``` bash
  java -jar target/Devoir-PROG2-1.0-SNAPSHOT.jar
```
